# 🌍 공유기 포트포워딩으로 외부 접속 설정 가이드

같은 와이파이가 아닌 **외부(LTE 등)에서도 접속**할 수 있도록 공유기 포트포워딩을 설정하는 방법입니다.

---

## 📡 네트워크 구조

```
외부 기기 (LTE) → 공인 IP:8888 → 공유기 → 내부 PC (192.168.35.20:8080)
```

---

## ✅ 설정 순서

### 1️⃣ 공인 IP 확인

아래 사이트에서 현재 공인 IP를 확인합니다.

```
https://whatismyip.com
```

예시: `YOUR_PUBLIC_IP`

> ⚠️ **반드시 `whatismyip.com`에서 확인하세요!**
> > PC 내부(`ipconfig` 등)에서 확인한 IP는 CGNAT 구조로 인해 실제 공인 IP와 다를 수 있습니다.
> > > 예) PC에서 보이는 IP: `162.120.184.40` (잘못된 주소 ❌)
> > > > 예) whatismyip.com에서 확인한 IP: `YOUR_PUBLIC_IP` (올바른 공인 IP ✅)
> > > >
> > > > ---
> > > >
> > > > ### 2️⃣ 공유기 관리 페이지 접속
> > > >
> > > > 브라우저 주소창에 입력:
> > > >
> > > > ```
> > > > http://192.168.35.1
> > > > ```
> > > >
> > > > > 안 되면 아래 주소 순서대로 시도해 보세요.
> > > > > > - `http://192.168.1.1`
> > > > > > - > - `http://192.168.0.1`
> > > > > >   >
> > > > > >   > - ---
> > > > > >   >
> > > > > >   > ### 3️⃣ 포트포워딩 메뉴 찾기
> > > > > >   >
> > > > > >   > 공유기 모델별 메뉴 위치:
> > > > > >   >
> > > > > >   > | 공유기 | 메뉴 경로 |
> > > > > >   > |--------|-----------|
> > > > > >   > | H724G (SK 브로드밴드) | 방화벽 → 포트 포워딩 |
> > > > > >   > | iptime | 고급 설정 → NAT/라우터 관리 → 포트포워드 설정 |
> > > > > >   > | 삼성/화웨이 | 네트워크 설정 → 포트포워딩 |
> > > > > >   >
> > > > > >   > ---
> > > > > >   >
> > > > > >   > ### 4️⃣ 포트포워딩 설정값 입력
> > > > > >   >
> > > > > >   > | 항목 | 값 |
> > > > > >   > |------|-----|
> > > > > >   > | 서비스 포트 | `8888` |
> > > > > >   > | 프로토콜 | `TCP` |
> > > > > >   > | 내부 IP 주소 | `192.168.35.20` |
> > > > > >   > | 포트 | `8080` |
> > > > > >   > | 설명 | `Tomcat WSL` |
> > > > > >   >
> > > > > >   > > ⚠️ 서비스 포트에 `8080` 입력 시 "이미 사용 중" 오류가 뜨면 `8888`, `18080`, `9080` 등 다른 포트를 사용하세요.
> > > > > >   > >
> > > > > >   > > **적용** 버튼 클릭 후 즉시 반영됩니다. (재부팅 불필요 ✅)
> > > > > >   > >
> > > > > >   > > ---
> > > > > >   > >
> > > > > >   > > ### 5️⃣ 외부에서 접속 테스트
> > > > > >   > >
> > > > > >   > > LTE 등 외부 네트워크에서 브라우저에 입력:
> > > > > >   > >
> > > > > >   > > ```
> > > > > >   > > http://YOUR_PUBLIC_IP:8888
> > > > > >   > > ```
> > > > > >   > >
> > > > > >   > > > `YOUR_PUBLIC_IP` 부분을 1단계에서 확인한 본인의 공인 IP로 변경하세요.
> > > > > >   > > >
> > > > > >   > > > ---
> > > > > >   > > >
> > > > > >   > > > ## ⚠️ 주의사항
> > > > > >   > > >
> > > > > >   > > > - **공인 IP는 재부팅 시 변경될 수 있습니다.**
> > > > > >   > > > -   고정 IP가 필요하면 DDNS 서비스(예: [DuckDNS](https://www.duckdns.org)) 사용을 권장합니다.
> > > > > >   > > >
> > > > > >   > > > -   - **192.168.x.x는 사설 IP**라 외부에서 직접 접속 불가합니다.
> > > > > >   > > >     -   캐시로 인해 잠깐 되는 것처럼 보일 수 있으니 공인 IP로 접속하세요.
> > > > > >   > > >  
> > > > > >   > > >     -   - 보안을 위해 사용하지 않을 때는 포트포워딩 규칙을 삭제하는 것을 권장합니다.
